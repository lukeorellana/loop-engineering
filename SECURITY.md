# Security Policy

## Supported Versions

This project is in early development. Security fixes are applied to the latest
release and the default branch.

## Reporting a Vulnerability

Please report security vulnerabilities privately rather than opening a public
issue.

- Use [GitHub private vulnerability reporting](https://github.com/lukeorellana/loop-engineering/security/advisories/new)
  to disclose a vulnerability.

Please include:

- A description of the vulnerability and its impact.
- Steps to reproduce or a proof of concept.
- Any known mitigations or workarounds.

We will acknowledge your report as soon as possible and keep you informed of
the remediation progress. Please do not publicly disclose the issue until a fix
has been released.
